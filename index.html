What is UniVo?

UniVo was born to break communication barriers.
It is an Augmentative and Alternative Communication (AAC) tool created to be accessible to everyone, without costs and without limitations. Our commitment is to inclusion and freedom: the code is open, collaborative, and can be used by anyone, anywhere.
